Documentación del Proceso de Preparación de Datos

1. Selección

El dataset fue generado a partir del archivo original osb_malnutricion5_17anos.csv, que contiene información antropométrica de niños y adolescentes entre 5 y 17 años en distintas regiones de Colombia. A partir de esta fuente, se seleccionaron las siguientes variables esenciales para análisis de malnutrición:

  • Peso (kg)
  • Altura (cm)
  • Lugar (ubicación geográfica del registro)

2. Identificación de fuentes de datos relevantes

La fuente contenía miles de registros. Se seleccionaron aleatoriamente 500 registros válidos como muestra representativa, conservando la estructura básica para el análisis exploratorio y desarrollo de modelos predictivos.

3. Limpieza

  • Se eliminaron filas con valores faltantes en las columnas de interés (peso, altura, lugar).
  • Se filtraron los datos con valores atípicos o no realistas:
    • Peso fuera del rango 10 kg - 150 kg
    • Altura fuera del rango 80 cm - 210 cm

4. Transformación

  • Se añadieron nuevas variables al dataset:
    • Edad: asignada aleatoriamente entre 5 y 17 años, para simular la edad del niño.
    • Sexo: asignado aleatoriamente como “Masculino” o “Femenino”.
  • Opcionalmente, se puede calcular una variable objetivo:
    • Estado Nutricional: usando el IMC (Índice de Masa Corporal) con la fórmula IMC = peso / (altura/100)^2 y clasificando como "Bajo peso", "Normal", o "Sobrepeso".

5. Documentación

Todos los pasos fueron documentados, incluyendo:

  • La fuente original (osb_malnutricion5_17anos.csv).
  • Selección de columnas relevantes y filtrado de registros válidos.
  • Asignación de variables adicionales (Edad, Sexo).
  • Exportación del dataset final en el archivo dataset_arbol_decision.csv, listo para usar en modelos como árboles de decisión o XGBoost.

Este proceso asegura que los datos estén preparados adecuadamente para el entrenamiento de modelos supervisados en tareas de clasificación de estado nutricional o análisis de factores asociados.
